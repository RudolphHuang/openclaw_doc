# Claude Code AINFT Provider 一键配置脚本

## 概述

一键配置脚本帮助用户快速将 AINFT 配置为 Claude Code 的模型提供商。脚本支持 Windows (PowerShell) 和 Linux/macOS (Bash) 两大平台，提供交互式引导，自动完成环境检查、API Token 配置、模型选择和配置文件更新。

## 工作流程

### 主流程图

```mermaid
flowchart TD
    Start([开始]) --> Init["初始化<br/>• 检测系统语言<br/>• 显示欢迎信息"]

    CheckEnv -->|检查失败| ErrorEnv["显示错误信息<br/>提示安装依赖"]
    ErrorEnv --> EndFail([结束])
    
    CheckEnv -->|检查通过| InputToken["输入 API Token<br/>• 格式校验<br/>• 确认提示"]
    
    InputToken --> FetchModels["获取模型列表<br/>• 调用 AINFT API<br/>• 认证验证"]

    FetchModels -->|获取失败| UseDefault["使用默认模型<br/>gpt-5-nano<br/>gpt-5-mini<br/>gpt-5"]
    
    UseDefault --> UpdateConfig
```

## 时序图

### 完整配置流程时序图

```mermaid
sequenceDiagram
    actor User as 用户
    participant Script as 配置脚本
    participant System as 系统环境
    participant AINFT as AINFT API
    participant Config as 配置文件
    
    User->>Script: 执行安装脚本
    Script->>System: 检测操作系统
    Script->>System: 检测 Shell 类型
    Script->>System: 检查 Claude 安装
    System-->>Script: 返回检查结果
    Script->>System: 检查 curl 安装
    System-->>Script: 返回检查结果
    
    alt 环境检查失败
        Script-->>User: 显示错误信息
    else 环境检查通过
        Script-->>User: 提示输入 API Token
        User->>Script: 输入 Token
        Script->>Script: 校验 Token 格式
        
        Script->>AINFT: GET /v1/models<br/>Authorization: Bearer Token
        
        alt API 调用成功
            AINFT-->>Script: 200 OK + 模型列表
            Script-->>User: 显示可用模型
            Script-->>User: 提示使用推荐模型
            
            alt 使用推荐模型
                User->>Script: 确认 (Y)
                Script->>Script: 设置推荐模型
            else 手动选择
                User->>Script: 拒绝 (N)
                User->>Script: 输入模型编号
                Script->>Script: 验证并设置模型
            end
        else API 调用失败
            AINFT-->>Script: 错误响应
            Script-->>User: 警告: 使用默认模型
            Script->>Script: 设置默认模型
        end
        
        Script->>Config: 读取现有配置
        Config-->>Script: 返回配置内容
        Script->>Config: 备份原配置
        Script->>Config: 写入新环境变量
        
        Script-->>User: 显示配置摘要
        Script-->>User: 提示重新加载配置
        Script-->>User: 显示测试命令
    end
```

## 使用方法

### Windows (PowerShell)

```powershell
# 方式一: 直接远程执行
iwr -useb https://chat.ainft.com/scripts/install-ainft-provider-claude.ps1 | iex

# 方式二: 下载后执行
.\install-ainft-provider.ps1
```

### Linux / macOS (Bash)

```bash
# 方式一: 直接远程执行
curl -fsSL https://chat.ainft.com/scripts/install-ainft-provider-claude.sh | bash

# 方式二: 下载后执行
chmod +x install-ainft-provider.sh
./install-ainft-provider.sh
```
